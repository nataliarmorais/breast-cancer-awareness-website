# 🎗 Breast Cancer Awareness Website

Website desenvolvido para conscientização sobre o câncer de mama, com foco em informação acessível e design responsivo.

---

## 🌐 Acesse o projeto

🔗 https://projeto-mama.vercel.app

---

## 💡 Objetivo

Criar uma página informativa e acessível para apoiar campanhas como o Outubro Rosa, promovendo conscientização e prevenção.

---

## 🛠️ Tecnologias

- HTML
- CSS
- SCSS

---

## 📱 Funcionalidades

- Layout responsivo
- Design adaptado para diferentes dispositivos
- Navegação simples e intuitiva

---

## 🎯 Destaque

Projeto focado em UI/UX e responsividade, com base em protótipo do Figma.
